# Ethics App Documentation

Welcome to the Ethics App documentation. This folder contains important information about the project structure, Supabase setup, and development guidelines.

## Documentation Files

- **setup-guide.md** - Step-by-step guide to set up your development environment
- **supabase-schema.md** - Database schema and table structures
- **api-reference.md** - Reference for all service methods and APIs
- **folder-structure.md** - Explanation of the project folder structure

## Quick Start

1. Copy `.env.example` to `.env` and fill in your Supabase credentials
2. Run `npm install` to install dependencies
3. Follow the setup guide for detailed instructions
4. Start developing with `npm start`

## Project Overview

This is a React Native application built with:
- **Expo** - For easy development and deployment
- **Supabase** - Backend as a Service (authentication, database, real-time)
- **React Navigation** - For navigation between screens
